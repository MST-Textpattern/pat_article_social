pat_article_social
==================

Textpattern CMS plugin.

Create social links and share counters for articles with meta Open Graph.
Support for major services: Twitter ®, facebook ®, G+ ®, Pinterest ® Tumblr ®, Instapaper ®, Pocket ®, Linkedin ® & Reddit ®.
